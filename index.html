Membuat Sistem Makanan Ringan

Proyek ini bertujuan untuk membuat dan mengelola database makanan ringan menggunakan REST API dan JWT. Pengguna dapat melakukan CRUD data makanan ringan dengan autentikasi aman.

Proyek KB & TK Menggunakan Vue.js

Sistem ini bertujuan untuk mempermudah pengelolaan data pendaftaran siswa, orang tua, serta pembayaran. Dibangun menggunakan Vue.js dan Laravel 10.

  • Formulir pendaftaran siswa yang responsif dan terintegrasi dengan local storage.
  • Backend Laravel dengan pendekatan MVC.
  • Tampilan modern dan mudah digunakan.
