Project Sistem Barang Makanan Ringan

Berikut adalah Hasil Membuat database Makanan ringan (CRUD)

Membuat Sistem Makanan ringan menggunakan Rest API dan JWT

Dalam proyek ini, saya bertanggung jawab untuk membuat dan mengelola database makanan ringan menggunakan Rest API (Representational State Transfer Application Programming Interface) dan JWT (JSON Web Tokens). Tujuan utama proyek ini adalah untuk menciptakan sistem yang memungkinkan pengguna untuk menelusuri, menambahkan, mengedit, dan menghapus data makanan ringan dengan aman dan efisien. Sebagai bagian dari pengembangan sistem manajemen inventaris, pencatatan makanan ringan merupakan bagian penting untuk dikelola. Dengan menggunakan Rest API, saya dapat menyediakan layanan yang dapat diakses melalui HTTP dan digunakan oleh berbagai aplikasi klien. Pengguna akan diotentikasi menggunakan JWT, memastikan hanya pengguna yang sah yang memiliki akses ke sistem.

Proyek KB & TK Menggunakan Vue.js

Dalam proyek ini, saya mengembangkan sistem manajemen data untuk TK Bintang Kasih menggunakan framework Vue.js. Proyek ini bertujuan untuk mempermudah proses pengelolaan data pendaftaran siswa, orang tua, serta pembayaran secara interaktif dan dinamis. Beberapa fitur utama dari proyek ini meliputi:

  • Formulir pendaftaran siswa yang responsif dan terintegrasi dengan local storage.
  • Menggunakan Laravel 10 untuk integrasi dengan database dan Menggunakan metode MVC
  • Desain yang modern dan ramah pengguna, sehingga calon siswa mudah untuk mendaftar.

Proyek ini tidak hanya membantu dalam pengelolaan data secara internal, tetapi juga memberikan kemudahan bagi orang tua dalam proses pendaftaran.
