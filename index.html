Project 2

Berikut ini adalah Hasil Desain Aplikasi berbasis Mobile menggunakan Flutlab dan database Menggunakan Google Firebase

Deskripsi Project Desain Aplikasi berbasis Mobile menggunakan Flutlab dan database Menggunakan Google Firebase

Sebagai bagian dari proyek UAS, saya memiliki kesempatan untuk mengembangkan sebuah aplikasi absensi mahasiswa berbasis mobile yang menggunakan Flutlab sebagai platform pengembangan utama dan database Google Firebase sebagai backend. Proyek ini menantang saya untuk menggabungkan kreativitas desain dengan fungsionalitas yang kuat, dan hasil akhirnya adalah sebuah aplikasi yang memenuhi tujuan proyek dengan baik. Tujuan utama dari proyek ini adalah untuk menciptakan sebuah solusi yang memudahkan pengelolaan absensi mahasiswa secara efisien dan efektif. Dengan aplikasi ini, pengajar dapat dengan mudah melakukan absensi di kelas menggunakan perangkat mobile mereka, dan data absensi secara otomatis disimpan ke dalam database Firebase untuk kemudian dianalisis lebih lanjut.

Project 3

Berikut ini adalah Hasil saya Membuat database Makanan ringan yang saya buat sendiri

Deskripsi Membuat database Makanan ringan menggunakan Rest API dan JWT

Dalam proyek ini, saya bertanggung jawab untuk membuat dan mengelola database makanan ringan menggunakan Rest API (Representational State Transfer Application Programming Interface) dan JWT (JSON Web Tokens). Tujuan utama proyek ini adalah untuk menciptakan sistem yang memungkinkan pengguna untuk menelusuri, menambahkan, mengedit, dan menghapus data makanan ringan dengan aman dan efisien. Sebagai bagian dari pengembangan sistem manajemen inventaris, pencatatan makanan ringan merupakan bagian penting untuk dikelola. Dengan menggunakan Rest API, saya dapat menyediakan layanan yang dapat diakses melalui HTTP dan digunakan oleh berbagai aplikasi klien. Pengguna akan diotentikasi menggunakan JWT, memastikan hanya pengguna yang sah yang memiliki akses ke sistem.
